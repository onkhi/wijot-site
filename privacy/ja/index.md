# wijot プライバシーポリシー

施行日: 2026年4月1日

## 1. 適用範囲

本プライバシーポリシーは、iOS アプリ「wijot」およびこれに付随するサポートページ等に適用されます。

## 2. wijot が扱う情報

### 2.1 メモ内容と設定

wijot はローカル中心のアプリです。  
現行実装では、メモ内容や一部設定は主として利用者の端末内、または Widget 表示のための共有保存領域に保存されます。

### 2.2 購入状態に関する情報

wijot は StoreKit を利用して、サブスクリプションまたは永久ライセンスの購入状態を確認します。  
決済そのもの、請求情報、支払い方法等は Apple が取り扱います。

### 2.3 お問い合わせ時に提供される情報

利用者がお問い合わせを行う場合、当方は、利用者が任意に提供した氏名、メールアドレス、問い合わせ内容、添付情報等を受け取る場合があります。

## 3. 利用目的

当方は、上記情報を以下の目的で利用します。

- アプリ機能の提供と維持
- Widget 表示との整合
- 購入状態の反映、購入復元、契約状態表示
- お問い合わせへの対応
- 不具合調査およびサポート品質の改善

## 4. 第三者サービス

### 4.1 Apple

App Store での配布、アプリ内課金、サブスクリプション管理、購入復元等には Apple のサービスが関与します。  
これらに関する情報の取扱いは、Apple の利用条件およびプライバシー方針にも従います。

### 4.2 解析広告 SDK

現時点では、独自の広告 SDK や解析 SDK は確認されていません。  
将来これが変わる場合は、本ポリシーを更新します。

## 5. 外部送信と保存

現時点では、メモ本文を当方サーバーへ同期・アップロードする独自機能は確認されていません。  
ただし、利用者がお問い合わせを行う場合や、Apple の購入基盤を利用する場合には、当該経路で必要な情報が処理されることがあります。

## 6. 利用者による管理

利用者は、必要に応じて以下を行えます。

- アプリ内メモの削除
- App Store を通じたサブスクリプション管理
- お問い合わせの停止

端末の故障、アプリ削除、バックアップ未実施等によりローカルデータが失われる可能性があるため、必要に応じて利用者自身で管理してください。

## 7. 改定

当方は、法令、機能変更、運営上の必要性等に応じて本ポリシーを改定することがあります。  
重要な変更がある場合は、適切な方法で周知します。
